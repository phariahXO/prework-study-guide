HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML attributes define additional information about an element

CSS

  • CSS is used to define the style of the webpage
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A declaration contains 2 important components: the CCS property we want to apply and the value of the property

Git

  • Git is a version control system that lets you manage and keep track of multiple versions of your project.
  • To acccess Git the command line is used
  • Git commands always start with "git" preceeding the command.
  • git status:checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • to switch to the main branch the command is "git checkout main"
  • to pull the remote branch to the local branch the command is "git pull origin main"

javascript

  • Javascript allows visitors to interact with the website
  • "API means Application Programming Interface
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A semicolon at the end of a line indicates where the statement ends(It is only requires when you need to seperate statements on a single line
  • JavaScript is CASE SENSITIVE
  • A variable can be given a value (In other words a name) (EX. myvariable= 'Bob')
  • Variables can hold values that have different data types
  • A "string" value must be indicated by single quote marks (EX. let myvariable= 'Bob';)
  • Number values don't have quotes around them
  • Boolean is a true/false value, these words are special keywords that don't need quote marks
  • An "array" is a structure that allows you to store multiple values in a single reference (EX. let myVariable= [1,"Bob","Steve",10]);
  • Everything in JavaScript is an "object" and can be stored in a variable
